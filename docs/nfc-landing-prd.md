---

## 1. 개요 (Overview)

### 1.1 프로젝트 배경
뮤즈드마레는 프랑스 샹파뉴에서 태어나 한국 심해에서 숙성되는 해저숙성 샴페인입니다. 각 병에 NFC 칩을 부착하여 고객이 스마트폰으로 탭하면, 해당 병의 여정과 진품 인증 정보를 담은 모바일 웹페이지로 연결됩니다.

### 1.2 프로젝트 목표
1. **진품 인증**: 블록체인 기반 위변조 불가능한 제품 이력 증명
2. **가치 전달**: 이중 떼루아(프랑스 토양 + 한국 바다)의 스토리텔링
3. **브랜드 경험**: 럭셔리 브랜드에 걸맞은 몰입형 디지털 경험
4. **신뢰 구축**: 숙성 과정의 투명한 데이터 공개

### 1.3 핵심 가치 제안
> "이 샴페인이 프랑스 떼루아에서 태어나 한국 바다에서 완성되기까지, 
> 변하지 않는 진실을 담은 디지털 여정"

## 2. 사용자 시나리오 (User Scenario)

### 2.1 기본 플로우
```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   1. 구매자가 병에 부착된 NFC 칩에 스마트폰을 탭                    │
│                        ↓                                        │
│   2. 고유 URL로 모바일 웹페이지 자동 연결                          │
│      (예: auth.musedemaree.com/bottle/MDM-2026-001)             │
│                        ↓                                        │
│   3. 블록체인 인증 확인 애니메이션                                 │
│                        ↓                                        │
│   4. 이 병의 고유 여정 스토리 표시                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 사용자 유형
| 유형 | 목적 | 주요 관심사 |
|------|------|------------|
| **구매자** | 진품 확인, 제품 스토리 감상 | 인증 여부, 숙성 데이터 |
| **선물 수령자** | 제품 가치 이해 | 브랜드 스토리, 희소성 |
| **와인 애호가** | 상세 정보 탐색 | 떼루아, 테이스팅 노트, 숙성 환경 |
| **컬렉터** | 소장 가치 확인 | 에디션 정보, 인양 기록, 희소성 |

## 부록 B: NFC 태그 URL 구조

```
https://auth.musedemaree.com/v/MDM-2026-001?sig=xxx&cnt=1

URL 구성:
├── Domain: auth.musedemaree.com
├── Path: /v/{bottleId}
│   └── bottleId: 고유 병 ID (MDM-YYYY-NNN)
└── Query Params:
    ├── sig: NTAG 424 DNA가 생성한 암호화 서명
    │       (탭할 때마다 변경됨 - 복제 방지)
    └── cnt: 스캔 카운터 (1부터 증가)

서버 검증 로직:
1. sig 파라미터를 NFC 마스터 키로 복호화
2. bottleId와 cnt가 매칭되는지 확인
3. cnt가 이전 스캔보다 큰지 확인 (리플레이 공격 방지)
4. 모든 검증 통과 시 verified: true 반환
```
