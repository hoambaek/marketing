# Sea Lab — 완도 해역 해양 데이터셋

> **Muse de Marée** 해저 숙성 샴페인 프로젝트의 해양 실측 데이터 아카이브
> Claude Science 전달용 데이터 문서 · 생성일 2026-07-02

---

## 1. 개요

Muse de Marée는 전남 완도 해역 수심 **30 m** 지점에서 샴페인을 해저 숙성하는 브랜드입니다.
"Sea Lab"(웹 경로 `/data-log`)은 이 숙성 해역의 환경 변수를 매일 수집·집계해 저장하는 데이터 로그입니다. 이 데이터셋은 UAPS(숙성 풍미 예측) 엔진의 계절·환경 보정 입력으로 사용됩니다.

| 항목 | 값 |
|------|-----|
| 관측 대상 해역 | 완도 해역 (전남 완도군, 보길도 인근) |
| 대표 좌표 | 위도 **34.31°N**, 경도 **126.76°E** |
| 기준 숙성 수심 | **30 m** |
| 데이터 저장소 | Supabase 테이블 `ocean_data_daily` (일별 집계) |
| 집계 단위 | 1일 1레코드 (시간별 원시 → 일 평균/최소/최대 집계) |
| 데이터 기간 | **2025-01-01 ~ 2026-07-01** (약 18개월) |
| 총 레코드 수 | **500 일** (기대 547일 중, 결측 47일) |
| 시간대 | Asia/Seoul (KST) 기준 수집 |

---

## 2. 데이터 출처 및 수집 파이프라인

데이터는 **두 소스의 하이브리드**로 구성됩니다.

### 2.1 KHOA — 국립해양조사원 (실측)

공공데이터포털 KHOA Open API를 통한 **현장 실측값**. 실측 우선 원칙(실측 > 모델 추정)에 따라, KHOA 값이 존재하면 Open-Meteo 모델값을 덮어씁니다.

| 관측소 | 코드 | 제공 항목 |
|--------|------|-----------|
| 완도 조위관측소 | `DT_0027` | 수온, 염분, 기압, 조위, 풍향/풍속, 유향/유속 |
| 완도항 부이 | `TW_0078` | 표층 유속·유향 (조류 실측) |
| 완도 조류예보 지점 | `20LTC03` | 당일 조류 창·낙조 유속 예보 |

- 일일 트래픽 한도: 10,000회
- 사용 서비스: `GetDTRecentApiService`(최신관측), `GetTWRecentApiService`(부이), `GetCrntFcstFldEbbApiService`(조류예보) 등

### 2.2 Open-Meteo (모델/재분석)

KHOA가 제공하지 않는 항목(파고·파주기·해류 모델값)과, KHOA 실측 공백 구간의 백필에 사용.

| API | 제공 항목 |
|-----|-----------|
| Marine API (`marine-api.open-meteo.com`) | 파고, 파향, 파주기, 해류 속도/방향, 표층 수온 |
| Weather API / Historical Archive | 표층 기압, 기온(2 m), 상대습도 |

> 7일 이전 과거 구간은 Historical Archive(ERA5 기반)로 조회.

### 2.3 병합 로직 (`data_source` 필드)

| `data_source` 값 | 의미 | 레코드 수 |
|------------------|------|-----------|
| `hybrid` | KHOA 실측 + Open-Meteo 병합 (수온·염분·조위·조류는 KHOA 실측) | **300** |
| `open-meteo` | Open-Meteo 단독 (KHOA 실측 공백 구간, 주로 2026-04 이후) | **200** |

염분·조위·조류는 **KHOA 실측이 있는 300일에만** 존재하며, Open-Meteo 단독 구간에는 결측입니다.

## 3. 스키마 / 데이터 딕셔너리

테이블 `ocean_data_daily` (일별 1레코드). DB는 snake_case, 단위는 아래와 같습니다.

| 컬럼 | 설명 | 단위 | 소스 | 커버리지(n/500) |
|------|------|------|------|------|
| `date` | 관측 일자 (PK, UNIQUE) | YYYY-MM-DD | — | 500 |
| `sea_temperature_avg` | 일 평균 수온 | °C | KHOA 우선 / Open-Meteo | 500 |
| `sea_temperature_min` | 일 최저 수온 | °C | 동일 | 500 |
| `sea_temperature_max` | 일 최고 수온 | °C | 동일 | 500 |
| `salinity` | 염분 | psu (‰) | **KHOA 실측** | 390 |
| `current_velocity_avg` | 해류 속도 (모델) | m/s | Open-Meteo | 206 |
| `current_direction_dominant` | 우세 해류 방향 | deg (0–360) | Open-Meteo | 206 |
| `wave_height_avg` | 일 평균 파고 | m | Open-Meteo | 206 |
| `wave_height_max` | 일 최대 파고 | m | Open-Meteo | 206 |
| `wave_period_avg` | 평균 파주기 | s | Open-Meteo | 200 |
| `surface_pressure_avg` | 표층 기압 | hPa | KHOA 우선 / Open-Meteo | 500 |
| `air_temperature_avg` | 기온 (2 m) | °C | Open-Meteo | 500 |
| `humidity_avg` | 상대습도 | % | Open-Meteo | 206 |
| `tide_level_avg` | 평균 조위 | cm | **KHOA 실측** | 390 |
| `tide_level_min` | 최저 조위 | cm | KHOA | 390 |
| `tide_level_max` | 최고 조위 | cm | KHOA | 390 |
| `tidal_current_speed` | 조류 유속 | cm/s | **KHOA 실측/예보** | 390 |
| `tidal_current_direction` | 조류 방향 | deg (0–360) | KHOA | 390 |
| `depth` | 숙성 수심 (상수) | m | — | 500 (전부 30) |
| `location_name` | 지점명 | — | — | "완도" / "완도 해역" |
| `latitude` / `longitude` | 좌표 | deg | — | 34.31 / 126.76 |
| `data_source` | 소스 구분 | enum | — | 500 |
| `created_at` / `updated_at` | 레코드 타임스탬프 | timestamptz | — | 500 |

> 참고: `salinity_records`(수동 염분 측정) 테이블은 별도로 존재하나 **현재 0건**. 염분은 모두 KHOA 실측(`salinity` 컬럼)에 병합돼 있습니다.

## 6. 데이터 품질 노트 ⚠️

Claude Science 분석 시 아래 사항을 반드시 고려하세요.

### 6.1 결측 (총 47일)

주로 **2025년 상반기 초기 수집 안정화 구간**에 집중되어 있습니다.

```
2025-02-20 ~ 2025-03-06 (15일 연속),
2025-04-17~18, 04-21~23, 04-27, 04-29~05-01, 05-04~05,
05-09, 05-13, 05-20, 05-22, 05-25~26,
2025-06-06~11, 06-15, 06-22, 06-29,
2025-07-04, 07-07~08, 07-13, 07-18, 2025-08-06
```

2025-08 이후로는 **일별 연속성이 거의 완전**합니다.

### 6.2 변수별 커버리지 불균형

- **수온·기압·기온**: 500일 전부 존재.
- **염분·조위·조류(KHOA)**: 390일. **2026-04-01부터 결측** — 이 구간은 KHOA 실측이 병합되지 않고 Open-Meteo 단독(`open-meteo`)으로 저장됨.
- **파고·파주기·해류·습도(Open-Meteo)**: 206일. 초기 구간(2025년 대부분)에 파고류 데이터가 비어 있음 — 파고 수집은 2025-12부터 본격화.

### 6.3 이상치 (확인 필요)

- `sea_temperature_min = 0.00`: 2025년 8·9·10월 일부 레코드에 여름철 **0°C 최저 수온**이 존재 → 물리적으로 불가능. 센서 결측이 0으로 기록된 것으로 추정. **일평균(`avg`)·일최고(`max`)는 정상**이므로 분석 시 `_min` 대신 `_avg` 사용 권장.
- `surface_pressure_avg = 898.10 hPa` (2025-01): 정상 해수면 기압(약 1013 hPa) 대비 비현실적으로 낮음 → 이상치 가능성.
- `tide_level_min = -23 cm`: 조위 음수는 정상 (약최저저조위 기준면 아래 간조 시 발생 가능).

### 6.4 소스 신뢰도

- 수온·염분·조위·조류는 **KHOA 현장 실측**(신뢰도 높음, 300일 hybrid 구간).
- 파고·파주기·해류 속도는 **Open-Meteo 모델 추정치**(실측 아님).
- 수심 정보(`depth=30`)는 관측이 아닌 **고정 상수**(숙성 지점 명세).
