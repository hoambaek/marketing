3. 4개 하위 점수 (월별 계산)

① 질감 성숙도 (textureMaturity)

equivalentLandYears = landAgingYears + (underseaMonths / (12 × effectiveTci))
effectiveTci = max(0.1, TCI / kineticFactor)

score = 100 / (1 + exp(-0.6 × (equivalentLandYears - 4.5)))

시그모이드 곡선: 0→100, 중점 4.5년

kineticFactor가 높으면 effectiveTci가 작아져 → 해저 숙성이 지상 대비 가속

현재: kf=1.0(고정) or 해류 유속 기반 동적 → 질감 꾸준히 상승

② 향 신선도 (aromaFreshness)

totalOxidationYears = landAgingYears + (underseaMonths × FRI / 12)

score = 100 × exp(-0.10 × totalOxidationYears)

지수 감쇠: 100에서 시작, FRI가 낮을수록 천천히 감소

FRI=0.56이면 해저 산화 속도가 지상의 56% → 향 보존 유리

현재: 꾸준히 하락 (시간이 갈수록 향 손실)

③ 기포 미세화 (bubbleRefinement)

rate = 0.08 × (BRI / 1.6) × depthFactor
depthFactor = min(depth / 30, 1.5)

score = 40 + (100 - 40) × (1 - exp(-rate × months))

로그 성장: 40에서 시작, 100으로 수렴

시간이 갈수록 기포가 미세화되지만, 포화됨

현재: 초기 급상승 → 포화

④ 환원취 리스크 (offFlavorRisk)

risk = baseRisk + maxAdditional / (1 + exp(-0.2 × (months - inflection)))

baseRisk: low=5, medium=10, high=20
inflection: low=22, medium=18, high=15

시그모이드: 시간이 지남에 따라 리스크 증가

질감 성숙도 > 70이면 최대 15% 리스크 감소 보정

현재 (low): 초기 5점 → 22개월 변곡점 → 점차 증가

4. 종합 품질 (compositeQuality)

가중 평균

base = texture × w.texture + aroma × w.aroma + bubble × w.bubble + (100-risk) × w.risk

기본 가중치: texture=0.30, aroma=0.30, bubble=0.20, risk=0.20

시너지 보너스 (최대 +15점)

avgScore > 60 AND harmony > 0.5 일 때:
synergyBonus = 15 × avgFactor × harmonyFactor

harmony = min(scores) / max(scores) — 4개 점수가 균형적일수록 보너스

TSI 보정

TSI ≥ 0.7: +3점 | TSI < 0.4: -3점 | 그 외: 0점

5. Soft Cap — 일직선의 원인

문제: 종합 품질이 ~9개월 이후 일직선

지상 기준 품질(landBaseline)은 m=0에서 계산됩니다:

landBaseline = compositeQuality(texture=m0, aroma=m0, bubble=40, risk=m0)
≈ 약 70점 (지상 숙성 기본값)

Soft Cap 적용:

delta = compositeQuality - landBaseline (= compositeQuality - 70)
cap = 20

if |delta| > 20:
  softDelta = 20 + log(1 + excess) × 3
  compositeQuality = 70 + softDelta

예시 (compositeQuality가 95점일 때):

delta = 95 - 70 = 25 (cap 20 초과)
excess = 25 - 20 = 5
softDelta = 20 + log(1+5) × 3 = 20 + 1.79 × 3 = 25.4
final = 70 + 25.4 = 95.4 → 거의 안 잘림

실제 문제: Soft Cap 자체는 잘 작동하지만, 질감↑과 향↓이 상쇄되면서 종합 품질이 자연스럽게 수렴하는 겁니다.

7. 왜 일직선인가 — 핵심 인사이트

원인 분석

  1. 질감↑ + 향↓ 상쇄: 질감(시그모이드 ↑)과 향(지수 감쇠 ↓)이 서로 상쇄되면서 가중평균이 일정 수준(~77-80점)에서 수렴
  2. 시너지 보너스 감소: 향이 하락하면 harmony(균형도)가 떨어져 시너지가 줄어듦
  3. 환원취 증가: 22개월 이후 환원취가 가속 상승 → 종합 품질 하락 압력
  4. 결과: 12~24개월 구간에서 종합 품질이 88~91점으로 수렴 → 사실상 동일

해양 프로파일이 반영 안 되는 이유

코드상 monthlyOceanProfiles가 전달되고 있지만:

  • 동적 FRI 값 범위가 좁음: 보정된 40m 수온이 6.5~15.7°C → FRI 0.3~0.8 → 고정값 0.56과 큰 차이 없음
  • 동적 BRI 값도 비슷: 수온 변화에 의한 BRI 변동폭이 작음
  • K-TCI 영향 제한적: kineticFactor가 질감에만 영향, 종합에서 30% 비중
  • 결론: 계절 변동이 반영되지만 4개 점수의 상쇄 효과로 종합 품질에는 미미한 차이

8. 개선 방향

A. 피크 표시 개선 (빠른 수정)

  • "Peak 23개월" 대신 "최적 구간 12~30개월" (품질 95% 이상 구간) 표시
  • 동일 품질이면 가장 빠른 도달 시점을 추천 (비용/시간 효율)

B. 계절 변동 가시화 (중간 수정)

  • 종합 품질에 계절 보정 (±2~3점) 추가: 겨울 투입 시 FRI 유리 → +2점
  • compositeQuality에 월별 미세 진동을 넣어 곡선에 변화 부여

C. 모델 근본 개선 (대규모)

  • 향 감쇠 모델을 비선형으로: 초기 급감 → 안정 구간 → 후기 급감 (3단계)
  • 환원취 모델에 계절 의존성 추가: 여름 고온 → 환원취 가속
  • Conservative Cap 완전 제거, 대신 개별 점수에 물리적 상한 적용
